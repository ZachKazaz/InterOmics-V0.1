**InterOmics V0.1 -- Local Setup Guide**
This guide explains how to run the InterOmics platform on your own computer.

The application currently has two parts:
- R Backend (runs analyses)
- React frontend (the web interface)
Both must be running at the same time.


**1. Install Required Software**

You must install these three programs first.

**Step 1 -- Install R**
Download and install R (version 4.3.3) from: https://cran.r-project.org/

**Step 2 -- Install RStudio**
Download and inst RStudio Desktop (version 4.3.3) from: https://posit.co/download/rstudio-desktop/

**Step 3 -- Install Node.js**
Download and install Node.js (LTS version) rom: https://nodejs.org/

After installing, open a terminal/command prompt and check installation:
"node -v"
"npm -v"

Both should print version numbers.


**2. Download the InterOmics repository**

**Option A - Download ZIP (easiest)**
1. Go to GitHub repository page
2. Click Code
3. Click Download ZIP
4. Unzip the folder somewhere on your computer

**Option B -- Clone with Git**
In Terminal: "git clone https://github.com/ZachKazaz/InterOmics-V0.1.git


**3. Install Frontend Dependencies**
Open a terminal (Command Prompt, PowerShell, or Mac Terminal).
Navigate to the project folder, then the frontend folder:
"cd InterOmics-V0.1"
"cd frontend"

Instal dependencies: 
"npm install"

This may take a few minutes.


**4. Install Required R Packages**

Open RStudio.
Open the InterOmics project folder in RStudio, then run:
"install.packages(c(
"plumber",
"jsonlite",
"readr",
"ggplot2",
"igraph",
"mixOmics", 
"tidyr",
"stringr", 
"purrr",
"reshape2",
"data.table",
"circlize"
))"

Wait for all packages to install.


**5. Start the R Backend**

In RStudio, open the file: "plumber.R"

Then run:
"library(plumber)
pr <- plumb("plumber.R")
pr$run(port = 8000)"

You should see something like:
"Running plumber API at http:// ...."

Leave this running.


**6. Start the Frontend**

Open a new terminal. 
Navigate to the frontend folder again: 
"cd InterOmics-V0.1/frontend"

Start the React app:
"npm run dev"

You should see something like:
"Local: http://localhost:1234"


**7. Open the App**

Open your web browser and enter the link given in terminal after starting React app (e.g. 'http://localhost:1234').

The InterOmics interface should load.


**Important Notes**
- The R backend must be running before running the frontend
- If something stops working, restart both:
(i) Stop R backend and rerun plumber
(ii) Stop frontend in terminal (Ctrl+C) and run "npm run dev" again
- Large datasets may take time to process
- All plots and tables are generated by the R backend


**Quick Start Summary**

If everything is already installed, the quick start is:

Start backend in R:
"pr <- plumb("plumber.R")
pr$run(port = 8000)"

Start frontend in Terminal:
"cd InterOmics-V0.1/frontend"
"npm run dev"

Open localhost link.
